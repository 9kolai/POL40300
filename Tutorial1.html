What is R?

Software environment for statistical computing and graphics and a programming language.

Why R?

R vs. other statistical software (SPSS, Stata, SAS)

  • Free and open source software.
  • More packages, more opportunities.
  • Commonly used in PolSci and other fields (many media fx.)
  • Large support community
  • Documenting your analysis
  • Great for data visualisation (ggplot2)
  • Steep learning curve, but worth it.
  • R is also a programming language (you can make it do almost anything)

R vs. other programming languages (Python):

  • More common in PolSci
  • Focus on data analysis and statistics

What is RStudio?

An IDE (Integrated Development Environment) for R

  • auto completing code
  • syntax highlighting
  • listing stored variables
  • and much more

Some good resources for today:

(I will also put these on Moodle.)

  • Erik Gahner Larsen and Zoltán Fazekas (2019): “Quantitative Politics with R” (http://qpolr.com/): Chapter 1 and 2 covers installation and basics of R.
  • Garrett Grolemund & Hadley Wickham: “R for Data Science” (https://r4ds.had.co.nz/): Sections 1.4 to 1.6 covers very briefly installation of R, basics of running code in R, and how to get help.
  • Chester Ismay and Albert Y. Kim (2019): “Statistical Inference via Data Science: A moderndive into R and the tidyverse” (https://moderndive.com/index.html): Chapter 1 covers installation, introduction to syntax, and help files in R.
  • Peng, Roger D. (2019): “R Programming for Data Science” (https://bookdown.org/rdpeng/rprogdatascience): History and background of R: Chapter 2; Installation: Chapter 3; Data types: Chapter 4; Loading and saving data: chapter 5.
